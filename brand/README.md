# Harmony Brand

These are the current brand assets for Harmony.

The direction is compact, serious, and tool-like: a strong monogram, restrained color, and a wordmark that feels more like a native developer product than a generic AI startup.

## Current Assets

- `assets/logo.svg` is the primary scalable symbol.
- `assets/wordmark.svg` is the primary scalable wordmark.
- `assets/wordmark-lockup.png` is the preferred preview lockup for public-facing docs.
- `assets/logo.png` and `assets/logo.webp` are raster logo exports.
- `assets/wordmark-wide.jpeg` is an alternate generated wordmark export.
- `fonts/` contains the current font source files and licenses.

## Direction

Harmony should avoid generic AI gradients, fake futuristic dashboards, and soft SaaS branding.

The brand should feel native, deliberate, local, and controlled.
